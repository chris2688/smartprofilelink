# 🔐 샘플 로그인 가이드

DB 연결 없이 앱을 테스트할 수 있는 샘플 계정입니다.

## 📋 샘플 계정 정보

### 계정 1: Test
```
이메일: test@example.com
비밀번호: password123
```

### 계정 2: Demo
```
이메일: demo@example.com
비밀번호: demo123
```

### 계정 3: Admin
```
이메일: admin@example.com
비밀번호: admin123
```

---

## 🚀 사용 방법

### 1. Flutter 앱 접속
```
http://localhost:8080
```

### 2. 로그인 화면으로 이동
- 온보딩 화면에서 "로그인" 클릭

### 3. 샘플 계정 선택
- 로그인 화면 하단에 **"샘플 계정으로 테스트하기"** 박스가 표시됩니다
- 원하는 계정을 클릭하면 자동으로 이메일/비밀번호가 입력됩니다

### 4. 로그인 버튼 클릭
- 1초 후 자동으로 메인 대시보드로 이동합니다

## 📝 샘플 로그인 작동 방식

1. **로그인 시도**
   - 이메일과 비밀번호를 샘플 계정과 비교
   - 일치하면 mock 토큰 생성하여 저장

2. **토큰 저장**
   - `mock_access_token_[타임스탬프]`
   - `mock_refresh_token_[타임스탬프]`
   - 로컬 저장소(SecureStorage)에 저장

3. **로그인 상태 유지**
   - 앱을 닫아도 토큰이 남아있음
   - 로그아웃 시 토큰 삭제
